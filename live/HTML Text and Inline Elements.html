HTML Tag Visualizer

A visual guide to common HTML text formatting tags with code samples.

Headings <h1> to <h6>

Heading tags define a hierarchy of titles and subtitles on a page. `<h1>` is the most important (and there should only be one per page), while `<h6>` is the least important.

h1: The most important heading

h2: Section heading

h3: Sub-section heading

h4: Deeper sub-section heading

h5: Even deeper heading
h6: The least important heading
<h1>h1: The most important heading</h1>
<h2>h2: Section heading</h2>
<h3>h3: Sub-section heading</h3>
<h4>h4: Deeper sub-section heading</h4>
<h5>h5: Even deeper heading</h5>
<h6>h6: The least important heading</h6>

Paragraph & Breaks <p>, <br>, <hr>

The `<p>` tag defines a paragraph. `<br>` creates a line break within text (use sparingly). `<hr>` creates a "horizontal rule," representing a thematic break in content.

This is a standard paragraph element, used for blocks of text. It automatically adds some space before and after itself. 

<p>This is a standard paragraph element...</p>

This paragraph contains a line break.
This text appears on a new line thanks to the <br> tag, but it's still part of the same paragraph. 

<p>This paragraph contains a line break.<br>This text appears on a new line...</p>

The line below is a <hr> (horizontal rule), used to create a thematic break between content. 

<p>...break between content.</p>
<hr>

Emphasis & Importance

`<strong>` indicates that text has strong importance. `<em>` places emphasis on text. While `<b>` (bold) and `<i>` (italic) look similar, they are purely for styling and carry no semantic meaning. `<u>` underlines text but should be avoided so it's not confused with a link.

very important, emphasized, bold, italics, and underlined. 

<strong>very important</strong>
<em>emphasized</em>
<b>bold</b>
<i>italics</i>
<u>underlined</u>

Text Formatting

`<small>` makes text smaller (e.g., fine print). `<sup>` and `<sub>` create superscript and subscript text. `<mark>` highlights text for reference. `<wbr>` provides an optional line-break opportunity in a very long word.

smaller text, E=mc2, H2O, highlighted, and pneumonoultramicroscopicsilicovolcanoconiosis. 

<small>smaller text</small>
E=mc<sup>2</sup>
H<sub>2</sub>O
<mark>highlighted</mark>
...sili<wbr>covolcanoconiosis

Semantic & Technical Tags

`<abbr>` defines an abbreviation, with the full term in a `title` attribute. `<dfn>` identifies the defining instance of a term. `<cite>` is used to reference the title of a creative work (e.g., a book or painting).

The W3C sets web standards. The dfn tag is used to indicate the defining instance of a term. The painting The Starry Night was created by Vincent van Gogh. 

<abbr title="World Wide Web Consortium">W3C</abbr>
<dfn>dfn</dfn>
<cite>The Starry Night</cite>

Code & Computer Input

`<code>` formats a snippet of computer code. `<pre>` preserves whitespace and line breaks, ideal for code blocks. `<kbd>` represents keyboard input. `<samp>` indicates sample output from a program. `<var>` defines a variable.

Blockquote <blockquote>

The `<blockquote>` tag is used for extended quotations that are cited from another source. Browsers typically render this as an indented block of text.

"The advance of technology is based on making it fit in so that you don't really even notice it, so it's part of everyday life."

— Bill Gates in The Road Ahead
<blockquote>
    <p>"..."</p>
    <footer>— Bill Gates in <cite>The Road Ahead</cite></footer>
</blockquote>
