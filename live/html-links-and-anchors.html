Section 1: Basic Links & Anchors

The navigation bar above uses anchor links. Clicking a link smoothly scrolls you to the corresponding section on this page. Each section has a unique id attribute, which the link's href attribute targets using a hashtag (#).

You can also link to external websites. This is an absolute link:

Section 2: The `target` Attribute

Sometimes, you don't want users to leave your page. The target="_blank" attribute tells the browser to open the link in a new tab or window.

Open MDN Docs in a New Tab

Note: For security reasons, it's good practice to also add rel="noopener noreferrer" when using target="_blank".

Section 3: Special Links

Links can do more than just navigate to web pages. The mailto: prefix in an href will open the user's default email client.

Contact Us via Email

Code for This Example 

<!-- Navigation link pointing to an internal anchor -->
<a href="#section1">Go to Section 1</a>

<!-- The section that the link above targets -->
<section id="section1">
  <h3>This is Section 1</h3>
</section>

<!-- Link to an external site that opens in a new tab -->
<a href="https://example.com" target="_blank">External Site</a>

<!-- A mailto link to open an email client -->
<a href="mailto:contact@example.com">Email Me</a>
